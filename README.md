# Employee Performance Analysis Dashboard

## Overview

This project is an **interactive Employee Analytics Dashboard** built using **Streamlit**.  
It allows users to analyze employee data such as **salary, experience, and performance ratings** using interactive visualizations.

The dashboard generates a sample employee dataset and provides multiple visualization options to explore the data.

---

## Features

- Interactive dashboard built with **Streamlit**
- Automatically generated **employee dataset**
- Multiple visualization options:
  - Employee Dataset Table
  - Salary by Employee (Bar Chart)
  - Performance Rating Distribution (Pie Chart)
  - Experience vs Salary (Line Chart)
  - Experience vs Salary (Scatter Plot)

## Installation

### 1. Install Required Libraries

```bash
pip install streamlit pandas numpy matplotlib seaborn
```
## How to Run the Project

Run the Streamlit app using:

```bash
streamlit run app.py

The dashboard will open automatically in your browser.
```

## Dashboard Visualizations

The dashboard includes the following views:

1. **Employee Dataset**  
   Displays the generated employee dataset in a table.

2. **Salary by Employee (Bar Chart)**  
   Shows salary distribution among employees.

3. **Performance Distribution (Pie Chart)**  
   Displays employee performance ratings.

4. **Experience vs Salary (Line Chart)**  
   Shows the relationship between experience and salary.

5. **Experience vs Salary (Scatter Plot)**  
   Compares employee experience and salary with performance ratings.
